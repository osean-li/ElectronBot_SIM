6.5 域随机化

为缩小仿真与真机的差距，在每次 env.reset() 时随机化物理参数：

def apply_domain_randomization(self):
    import random

    for i in range(self.model.njnt):
        self.model.damping[i] *= random.uniform(0.8, 1.2)      # 阻尼 ±20%
        self.model.frictionloss[i] *= random.uniform(0.5, 1.5) # 摩擦 ±50%

    for i in range(self.model.nbody):
        self.model.body_mass[i] *= random.uniform(0.85, 1.15)  # 质量 ±15%

    for i in range(self.model.nu):
        self.model.actuator_gain[i] *= random.uniform(0.9, 1.1) # 增益 ±10%

随机化参数范围：

参数 范围 模拟的物理差异
关节摩擦 ±50% 3D 打印公差 + 润滑差异
关节阻尼 ±20% 舵机齿轮阻尼
执行器增益 ±10% 舵机个体差异
零件质量 ±15% PLA 密度不均匀
观测噪声 0-0.5° 无编码器开环漂移

效果对比：

训练方式 仿真成功率 真机迁移成功率
无域随机化 95% ~40%
有域随机化 92% ~70-80%
